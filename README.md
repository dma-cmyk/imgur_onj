# Imgur ONJ React 🚀

おんJ民による、おんJ民のための、最強Imgurアップローダー。
React + TypeScript + Vite + Gemini AI で生まれ変わった、次世代の画像共有ツールです。

## 💠 主な機能

### 🛠 多彩なアップロード
- **ファイル選択**: ドラッグ＆ドロップにも対応。
- **クリップボード**: コピーした画像をそのまま貼り付け。
- **タブ撮影**: 今見ているページをワンクリックでスクショ。
- **動画キャプチャ**: デスクトップやウィンドウを録画して共有。
- **URL指定**: ネット上の画像を直接Imgurへ転送。

### 🤖 Gemini AI 連携（要APIキー）
- **プライバシー保護**: 名前や住所、顔をAIが自動検知。アップロード前にモザイクをかけられます。
- **自動タグ付け**: 画像の内容をAIが分析し、最適なタグを3〜5個自動生成。
- **自動タイトル・説明文**: Imgur側にAI生成のタイトルと説明を自動入力。管理が劇的に楽になります。
- **セマンティック検索**: 「猫」「笑顔」など、AIが考えた説明文の内容から画像を検索可能。

### ☘️ おんJ特化機能
- **おんJダイレクト貼り付け**: 履歴のボタンを押すだけで、おーぷん2chの書き込み欄にURLを魔法のように自動入力。
- **サイドパネル最適化**: Chromeのサイドパネルにぴったりフィットするボーダレスデザイン。
- **100% 日本語UI**: 日本語掲示板での利用に最適な、親しみやすいインターフェース。

## 🚀 インストール方法

1. このリポジトリをクローンまたはダウンロードします。
2. `npm install` で依存関係をインストールします。
3. `npm run build` を実行して、`dist` フォルダを生成します。
4. Chromeで `chrome://extensions/` を開きます。
5. 「デベロッパーモード」をONにします。
6. 「パッケージ化されていない拡張機能を読み込む」から、本プロジェクトの **`dist`** フォルダを選択してください。

## ⚙️ 設定

1. 設定画面（⚙️アイコン）を開きます。
2. **Imgur Client ID** を入力します。
3. AI機能を使う場合は、[Google AI Studio](https://aistudio.google.com/app/apikey) で取得した **Gemini APIキー** を入力してください。
4. 「モデルリストを更新」ボタンを押し、最新のモデル（Gemini 2.0 Flash等）を選択して保存してください。

- **AI**: Google Gemini API (Generative AI SDK)
